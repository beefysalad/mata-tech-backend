# Security Policy

## Supported Versions

This project does not have a formal release process. The default branch is considered the active development version.

## Reporting a Vulnerability

If you discover a security issue, please report it privately by contacting the maintainers. Do not open a public issue with sensitive details.

When reporting, please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fixes (if known)

We will acknowledge receipt and provide updates on remediation progress.
